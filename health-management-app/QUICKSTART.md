# Quick Start Guide - Health Management App

## Getting Started in 3 Steps

### Step 1: Install Dependencies
```bash
cd health-management-app
npm install
```

### Step 2: Run Development Server
```bash
npm run dev
```

### Step 3: Open Browser
Navigate to `http://localhost:3000`

That's it! The app will open automatically in your browser.

## What You'll See

- **Dashboard**: Overview of all your health metrics
- **Vitals**: Track heart rate, blood pressure, weight, and sleep
- **Medications**: Manage your medications and schedules
- **Appointments**: Schedule and track doctor appointments
- **Activities**: Log workouts and physical activities
- **Goals**: Set and track health goals

## Sample Data

The app comes pre-loaded with sample data to help you get started. You can:
- Add new entries using the "+" buttons
- Delete entries using the "X" buttons
- All data is saved automatically to your browser

## Technologies Used

- React 18
- Tailwind CSS
- Recharts (Charts)
- Lucide React (Icons)
- Vite (Build Tool)
